# Web Scraping 2024 - 2025

Purtroppo le directory sono in ordine **alfabetico**, quindi devi fare dei *magheggi* per metterli in ordine

Il formato delle directory è: `LezioneXXYYZZ`, dove:
- **XX** è il giorno
- **YY** è il mese
- **ZZ** è l'anno

Mancano due lezioni, ma non posso caricarle qui perchè contengono DataSets troppo grandi
